<h1>如何安装Ubuntu</h1>
<h2>关闭 Windows 磁盘加密</h2>
关闭 Windows Bitlocker 加密和 Sercure Boot： 需要关闭后者才能安装除了 Windows 以外的操作系统，但 Windows 自带 Bitlocker 文件加密，若直接关闭，会导致无法打开系统。
<h2>关闭安全启动</h2>
关闭Sercure Boot：不同品牌按键不一样。
